A responsive landing page designed to help students discover internship opportunities and start their professional journey.

The website includes a hero section, navigation bar, feature cards, and responsive design for different screen sizes.

## Features

- Responsive navigation bar
- Hero section with background image
- Feature cards layout
- Smooth scrolling
- Modern UI design

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Responsive Design
- Google Fonts

## How to Run

1. Clone the repository
2. Open the project folder
3. Run with Live Server or open `index.html`
